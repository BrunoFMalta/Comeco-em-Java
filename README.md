# 🚀 Começo em Java

Este repositório acompanha minha evolução no aprendizado de **Java**, através do curso do professor [Nelio Alves](https://www.udemy.com/course/java-curso-completo/).  
Aqui registro cada exercício, projeto e tecnologia nova que exploro durante meus estudos.

---

## 📚 Objetivos
- Consolidar fundamentos da linguagem Java
- Praticar lógica de programação com exercícios progressivos
- Documentar minha jornada de aprendizado
- Compartilhar conhecimento e trocar ideias com a comunidade

---

## 🛠️ Tecnologias utilizadas
- **Java SE** (versão recomendada pelo curso)
- **IDE**: Eclipse / IntelliJ (dependendo do exercício)
- **GitHub** para compartilhamento

---

## 📂 Estrutura do repositório
- `exercicios-basicos/` → primeiros passos e lógica fundamental
- `orientacao-objetos/` → classes, objetos, herança, polimorfismo
- `projetos/` → exercícios mais completos e desafiadores
- `novas-tecnologias/` → experimentos além do curso
